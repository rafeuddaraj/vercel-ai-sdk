আজকের এই ভিডিওতে আমরা শুরু করছি আমাদের AI Journey এর একদম প্রথম লেসন।
এই কোর্সে আপনি শিখবেন কীভাবে Next.js ব্যবহার করে বাস্তব, প্রোডাকশন-রেডি AI powered application তৈরি করা যায়।
তবে কোড লেখার আগে কিছু জিনিস পরিষ্কারভাবে বোঝা খুব জরুরি।
কারণ ভিত্তি ঠিক না হলে পরের সব লেসন বুঝতে সমস্যা হবে।

কিছু বছর আগেও যদি আপনি আপনার অ্যাপে AI ফিচার যোগ করতে চাইতেন, তাহলে ব্যাপারটা মোটেও সহজ ছিল না।
ডেটা সায়েন্স টিম, মেশিন লার্নিং ইঞ্জিনিয়ার, বিশাল ইনফ্রাস্ট্রাকচার আর অনেক সময় ও টাকা দরকার হতো।
কিন্তু এখন সময় বদলে গেছে।
আজ একজন ওয়েব ডেভেলপার হিসেবেই আপনি এমন AI ফিচার বানাতে পারেন, যেগুলো আগে শুধু বড় কোম্পানিগুলোই বানাতে পারত।

আজ আপনি এমন অ্যাপ তৈরি করতে পারেন যেটা মানুষের মতো লিখতে পারে, প্রশ্নের উত্তর দিতে পারে, কনটেক্সট বুঝতে পারে, এমনকি ছবি বা কোড বিশ্লেষণও করতে পারে।
আর সবচেয়ে গুরুত্বপূর্ণ কথা হলো, এসব করতে আপনার কোনো Machine Learning বা Data Science ব্যাকগ্রাউন্ড থাকার দরকার নেই।

এই কোর্সে আমরা মূলত দুইটি শক্তিশালী টুল ব্যবহার করবো।
প্রথমটি হলো Next.js।
আপনি যদি React নিয়ে কাজ করে থাকেন, তাহলে Next.js আপনার কাছে নতুন কিছু নয়।
এটা আমাদের আধুনিক ওয়েব অ্যাপ বানানোর জন্য প্রয়োজনীয় সবকিছু এক জায়গায় দেয়।
Routing, API, Server, Client সবকিছু একই ফ্রেমওয়ার্কের ভেতরে।

দ্বিতীয় এবং সবচেয়ে গুরুত্বপূর্ণ টুলটি হলো Vercel এর AI SDK।
এই AI SDK মূলত AI integration কে অনেক সহজ করে দেয়।
সরাসরি OpenAI বা অন্য কোনো AI provider এর API ব্যবহার করলে অনেক জটিলতা থাকে।
AI SDK সেই জটিলতাগুলো নিজে সামলে নেয়, যাতে আপনি শুধু আপনার অ্যাপের লজিক আর ইউজার এক্সপেরিয়েন্স নিয়ে ভাবতে পারেন।

AI SDK তিনটি প্রধান অংশে ভাগ করা।
একটি হলো Core অংশ, যেখানে text generate করা, streaming response, structured output এসব কাজ করা যায়।
আরেকটি হলো UI অংশ, যেখানে React এর জন্য তৈরি করা কিছু বিশেষ hook আছে, যেগুলো দিয়ে খুব সহজে chat বা streaming UI বানানো যায়।
আর তৃতীয় অংশটি হলো React Server Components এর জন্য, যেটা এখনো experimental। এই কোর্সে আমরা সেটি ব্যবহার করবো না।

এই কোর্সটি মূলত তাদের জন্য তৈরি, যারা Next.js জানেন কিন্তু AI নিয়ে কাজ করা শুরু করবেন কীভাবে সেটা পরিষ্কার না।
আপনি যদি JavaScript এর বেসিক বিষয়গুলো জানেন, React এর fundamentals বুঝেন, আর Next.js এর App Router সম্পর্কে ধারণা থাকে, তাহলে এই কোর্সটি আপনি নিশ্চিন্তে করতে পারবেন।
TypeScript জানা থাকলে ভালো, না জানলেও সমস্যা নেই। কোর্স চলাকালেই প্রয়োজনীয় অংশগুলো পরিষ্কার হয়ে যাবে।

এখন চলুন development environment setup এর কথায় আসি।
এটাই Lesson 1 এর সবচেয়ে গুরুত্বপূর্ণ practical অংশ।

প্রথমেই আপনাকে একটি নতুন Next.js অ্যাপ তৈরি করতে হবে।
টার্মিনাল ওপেন করে `create-next-app` ব্যবহার করে একটি প্রজেক্ট তৈরি করবেন।
এখানে আমরা TypeScript ব্যবহার করবো, Tailwind CSS রাখবো, এবং App Router সিলেক্ট করবো।
কারণ এগুলোই বর্তমানে modern Next.js development এর standard।

একবার প্রজেক্ট তৈরি হয়ে গেলে, পরের ধাপ হলো প্রয়োজনীয় dependency install করা।
আমরা AI SDK এর core package install করবো, OpenAI provider install করবো, এবং React এর জন্য AI SDK এর আলাদা package ব্যবহার করবো।
এছাড়া Zod install করবো, যেটা পরের লেসনগুলোতে type-safe validation এর কাজে লাগবে।

এরপর আসে সবচেয়ে সংবেদনশীল অংশ, OpenAI API Key।
AI model ব্যবহার করা সম্পূর্ণ ফ্রি না, এটা আপনাকে মাথায় রাখতে হবে।
আপনাকে OpenAI তে একটি account তৈরি করতে হবে, billing add করতে হবে, এবং তারপর একটি secret API key generate করতে হবে।
এই API key আসলে আপনার অ্যাপের জন্য AI ব্যবহারের অনুমতি।

এই key কখনোই সরাসরি কোডের মধ্যে লেখা যাবে না।
আমরা `.env.local` ফাইল ব্যবহার করে API key রাখবো।
এই ফাইলটি Next.js নিজে থেকেই GitHub থেকে exclude করে রাখে, ফলে আপনার key নিরাপদ থাকে।

সব স্টেপ ঠিকভাবে শেষ হলে, আপনি dev server চালু করবেন।
`npm run dev` কমান্ড চালিয়ে ব্রাউজারে `localhost:3000` ওপেন করবেন।
যদি Next.js এর welcome screen দেখতে পান, তাহলে বুঝবেন আপনার setup পুরোপুরি ঠিক আছে।

এই Lesson 1 এ আমরা কোনো AI feature বানাইনি।
কিন্তু এই লেসনটাই পুরো কোর্সের ভিত্তি।
কারণ এখানেই আপনি বুঝে নিলেন AI কীভাবে accessible হয়েছে, কোন টুল ব্যবহার করবেন, আর কীভাবে নিরাপদভাবে কাজ শুরু করবেন।

এই লেসন শেষে আপনার মাথায় এটা পরিষ্কার থাকা উচিত যে
AI এখন আর দূরের কোনো বিষয় না,
Next.js এবং AI SDK ব্যবহার করে আপনি নিজেই শক্তিশালী AI powered application বানাতে পারেন,
এবং আপনি এখন পুরোপুরি প্রস্তুত পরের লেসনে আসল কাজ শুরু করার জন্য।

পরের লেসনে আমরা আমাদের প্রথম AI powered feature তৈরি করবো।
আপনি AI কে প্রশ্ন করবেন, আর AI আপনাকে উত্তর দেবে।
সেখান থেকেই আমাদের আসল journey শুরু হবে।
