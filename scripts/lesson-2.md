এই লেসনে আমরা একদম হাতে কলমে আমাদের ডেভেলপমেন্ট এনভায়রনমেন্ট তৈরি করবো। আগের লেসনে আমরা বুঝেছি Next.js আর Vercel AI SDK দিয়ে কী ধরনের AI অ্যাপ বানানো যায় এবং কেন এই টুলগুলো ব্যবহার করা হচ্ছে। এবার সেই ধারণাগুলো বাস্তবে কাজে লাগানোর সময়।

আপনি যদি আগে থেকেই Next.js ব্যবহার করে থাকেন, তাহলে অনেক কিছুই পরিচিত লাগবে। আর যদি নতুন হয়ে থাকেন, তবুও কোনো সমস্যা নেই। আমি খুব ধীরে ধীরে সব কিছু ব্যাখ্যা করবো।

চলুন শুরু করি।

সবার প্রথমে আমাদের একটি নতুন Next.js অ্যাপ তৈরি করতে হবে। এর জন্য আপনার কম্পিউটারে Node.js ইনস্টল থাকতে হবে। যদি আগে থেকেই ইনস্টল করা থাকে, তাহলে টার্মিনাল বা কমান্ড প্রম্পট খুলুন। তারপর সেখানে লিখুন
`npx create-next-app@latest`
এরপর আপনার প্রজেক্টের একটি নাম দিন। ধরুন আমরা নাম দিলাম `next-ai-app`।

এখন কিছু প্রশ্ন আসবে। এখানে আপনি যেগুলো সিলেক্ট করবেন, আমি একে একে বলছি।
TypeScript এর জন্য আপনি Yes সিলেক্ট করবেন, কারণ আমরা টাইপ সেফ কোড লিখতে চাই।
ESLint এর জন্য Yes, কারণ এটা আমাদের কোড পরিষ্কার রাখতে সাহায্য করে।
Tailwind CSS এর জন্যও Yes, কারণ UI বানাতে এটা খুব সহজ আর দ্রুত।
Source directory এর জন্য Yes।
App Router এর জন্য Yes, কারণ আমরা নতুন App Router ব্যবহার করবো।
TurboPack এর জন্য No রাখতে পারেন, চাইলে পরে এক্সপেরিমেন্ট করবেন।
Import alias এর জন্য No।

এগুলো সিলেক্ট করার পর কিছুক্ষণ সময় নেবে এবং আপনার একটি ফ্রেশ Next.js প্রজেক্ট তৈরি হয়ে যাবে।

এখন টার্মিনালে লিখুন
`cd next-ai-app`
মানে আমরা প্রজেক্ট ফোল্ডারের ভেতরে ঢুকছি।

এইবার সবচেয়ে গুরুত্বপূর্ণ অংশ। আমাদের AI SDK ইনস্টল করতে হবে। Vercel এর AI SDK খুব সুন্দরভাবে ভাগ করা, তাই আমরা শুধু যেগুলো দরকার সেগুলোই ইনস্টল করবো।

টার্মিনালে লিখুন
`npm install ai @ai-sdk/openai @ai-sdk/react zod`

এখানে একটু ব্যাখ্যা করি।
`ai` হলো core SDK, যেখানে সব মূল ফাংশন থাকে।
`@ai-sdk/openai` আমাদের OpenAI এর মডেলের সাথে কানেক্ট করে।
`@ai-sdk/react` React এর জন্য কিছু রেডি হুক দেয়, যেগুলো পরে আমরা ব্যবহার করবো।
আর `zod` ব্যবহার করা হয় টাইপ সেফ ভ্যালিডেশন করার জন্য, যেটা আমরা পরে শিখবো।

ইনস্টল শেষ হলে আমরা পরের ধাপে যাবো।

এখন আমাদের একটি OpenAI API Key লাগবে। এর জন্য আপনাকে যেতে হবে OpenAI এর ওয়েবসাইটে। সেখানে গিয়ে একটি অ্যাকাউন্ট তৈরি করুন। এরপর API Keys সেকশনে গিয়ে একটি নতুন Secret Key জেনারেট করুন।

এখানে একটা সত্য কথা আপনাকে জানানো জরুরি। AI মডেল ব্যবহার করতে টাকা লাগে। এই কোর্স ফলো করতে হলে আপনাকে বিলিং সেটআপ করতে হবে। সাধারণত ১০ থেকে ২০ ডলার যথেষ্ট হয় শেখার জন্য। আমার মতে এটা খুব ভালো ইনভেস্টমেন্ট।

যখন আপনার API Key তৈরি হয়ে যাবে, তখন সেটা কোথাও হার্ডকোড করা যাবে না। এটা খুব গুরুত্বপূর্ণ।

এখন প্রজেক্টের রুট ফোল্ডারে একটি নতুন ফাইল তৈরি করুন। নাম হবে `.env.local`
এই ফাইলের ভেতরে লিখুন
`OPENAI_API_KEY="আপনার_API_KEY_এখানে"`

মনে রাখবেন, এই ফাইল Git এ যাবে না। Next.js নিজে থেকেই `.env.local` কে gitignore করে রাখে। তাই আপনার API Key নিরাপদ থাকবে।

এই পর্যন্ত হলে আমাদের সেটআপ প্রায় শেষ।

এখন টার্মিনালে লিখুন
`npm run dev`

ডেভেলপমেন্ট সার্ভার চালু হয়ে যাবে। ব্রাউজারে যান
`http://localhost:3000`

আপনি যদি Next.js এর ওয়েলকাম পেজ দেখতে পান, তাহলে বুঝবেন সব কিছু ঠিকভাবে কাজ করছে।

এই লেসনে আমরা যা করেছি সেটা খুবই গুরুত্বপূর্ণ। আমরা একটি নতুন Next.js প্রজেক্ট তৈরি করেছি, প্রয়োজনীয় AI SDK ইনস্টল করেছি, OpenAI API Key সেট করেছি এবং নিশ্চিত হয়েছি যে সবকিছু ঠিকভাবে রান করছে।

এই সেটআপটাই আমাদের পুরো কোর্সের বেস। পরের লেসন থেকে আমরা আসল কোড লেখা শুরু করবো। আমরা শিখবো কীভাবে OpenAI এর সাথে কথা বলা যায়, কীভাবে AI থেকে টেক্সট জেনারেট করা যায় এবং ধীরে ধীরে একটি বাস্তব AI ফিচার তৈরি করবো।

তাই যদি কোথাও সমস্যা হয়ে থাকে, এই লেসনটা আবার দেখুন, নিজের হাতে টাইপ করে করুন। কারণ এই ভিত্তি শক্ত হলে পরের সব লেসন অনেক সহজ লাগবে।

পরের লেসনে আমরা লিখবো আমাদের প্রথম AI কোড।
